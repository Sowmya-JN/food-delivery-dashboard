Food Delivery Performance Dashboard

Project Overview
This project is an end-to-end Data Analytics dashboard created using Excel, MySQL, and Power BI. The dashboard analyzes food delivery operations and provides insights into customer satisfaction, delivery performance, and order distribution.

Tools Used
- Excel
- MySQL
- Power BI

Features
- Total Orders KPI
- Average Delivery Time
- Average Customer Rating
- Orders by Location
- Delivery Time Distribution
- Customer Feedback Analysis
- Interactive Filters and Slicers

SQL Analysis
SQL queries were used to analyze:
- Average ratings
- Delivery performance
- Location-wise order analysis
- Discount impact on customer ratings

Objective
The objective of this project is to analyze food delivery performance and create an interactive dashboard for business insights and decision-making.

Skills Demonstrated
- Data Cleaning
- SQL Query Writing
- Data Visualization
- Dashboard Design
- Business Insights Generation
